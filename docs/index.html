Do Heart Disease Mortality Rates Differ When NASSA Prescription Trends Increase?

B220774

Introduction

Mental health and heart disease are prominent public health challenges within Scotland@. The recent 2022 census revealed that the percentage of individuals reporting mental health conditions, conditions that affect your well-being, increased from 4.4% in 2011 to a staggering 11.3%, denoting the greatest change across all health categories. Heart disease describes the range of conditions impeding heart activity and structure, with coronary heart disease occurring most frequently. Since the early 2000s, cardiovascular disease mortality has reduced, however records depict a recent incline @. The study by (Alot?) found that individuals with severe mental illness, including schizophrenia, bipolar disorder and sever depression, demonstrated higher risk for cardiovascular disease resulting in premature death. Antidepressants treat multiple mental health conditions, however, the noradrenergic and specific serotonergic antidepressant (NASSA) Mitrazapine is used in the UK as a follow-up treatment if first-line SSRIs fail to work. Therefore, prescription trends can offer valuable insights into depression trends. This assessment explores the intersection of NASSA usage and heart disease mortality in Scotland by visualizing trends across health boards and age groups. It initially highlights that general antidepressant prescriptions have risen while in-patient mental health admissions reduced. Additionally, that mortality rates are increasing over time, with variations in NASSA usage across health board regions potentially correlating with increased death rates. Without conducting analysis, this report presents a cohesive and detailed overview of the data, offering foundation for future exploration of these critical public health issues.

Antidepressant Data Preparation

In this next section, the antidepressant data is imported and wrangled before the combined plot can be created. The antidepressants selected include; Mirtazapine, a oradrenaline and specific serotonergic antidepressant (NASSA), Fluoxetine, a selective serotonin reuptake inhibitors (SSRI), Venlafaxine, a serotonin-noradrenaline reuptake inhibitors (SNRIs) and Amitriptyline, a tricyclic antidepressants (TCAs).

### Data Import

#To import antidepressant prescription information from 2015 to 2023, dataset csv files of the months October, November, December, are downloaded from https://www.opendata.nhs.scot/dataset/prescriptions-in-the-community and added to the folder named "WinterDrugs2015-2023". This is to ensure the time frame of data collected was maximised. Prescription data for 2015 was only available during these months.

# All CSV files listed in the specified directory
ADFiles <- list.files(here("docs","WinterDrugs2015-2023"), pattern = "csv")

# Function defined to read and process each CSV file cohesively
ReadADFiles <- function(file_name) {
  # Extract the year from the file name using lubridate package in tidyverse. This enables plotting with Year as a variable
  year <- str_extract(file_name, "\\d{4}") %>% as.numeric()
  # Read the CSV file, process the data add year column
  data <- read_csv(here("docs","WinterDrugs2015-2023", file_name), show_col_types = FALSE,  col_types = cols(.default = "c")) %>% # Convert all columns to character type to avoid warnings due to inconsistent data types
    mutate(Year = year, PaidQuantity = as.numeric(PaidQuantity)) %>% # Specifically convert paid quantity to numeric format because the data contains numbers, allows for mathematical operations (e.g sum(), mean())
#Filter for specific antidepressants
    filter(str_detect(BNFItemDescription, "MIRTAZAPINE|FLUOXETINE|VENLAFAXINE|AMITRIPTYLINE"),) %>%
    return(data)
}
# This applies the ReadADFiles function to all the CSV files and combines the results, the large amount of data
ADs_Df <- ADFiles %>%
  map_dfr(ReadADFiles)
### Data Wrangling

# The data is cleaned again using janitor, and transformed  
Clean_ADs_Df <- ADs_Df %>%
# HBT2014 and HBT columns are combined into a new column called HB, prioritising data from the HBT column as this represents the most recent datasets
  mutate(HB = coalesce(HBT, HBT2014), 
#Antidepressants are categorised based on BNFItemDescription which is the British National Formulary information on medicines administered in the United Kingdom.
         Antidepressant = case_when(
             str_detect(BNFItemDescription, "MIRTAZAPINE") ~ "Mirtazapine(NASSA)",
             str_detect(BNFItemDescription, "FLUOXETINE") ~ "Fluoxetine(SSRI)",
             str_detect(BNFItemDescription, "VENLAFAXINE") ~ "Venlafaxine(SNRI)",
             str_detect(BNFItemDescription, "AMITRIPTYLINE") ~ "Amitriptyline(TCA)",
             TRUE ~ NA_character_  # Leave as NA if it doesn't match any MH Drug
           )) %>%  # Create 'hb' from HBT2014 or hbt
  select(-HBT2014, -HBT, HB, Year, PaidQuantity) %>% # select only relevant columns
  filter(!is.na(PaidQuantity)) # ensure there are no "NA" cells in the "PaidQuantity" column which is important in preventing inaccuracies in calculations or summaries, this could also lead to incomplete or misleading visualisations

In this section, population data csv file is downloaded and obtained using here(). Data visualisation to view Mitrazipine prescriptions per person differences across Scottish healthvboards. This information will be used to determine if there is a difference in the heart disease mortality statistics when prescriptions are high versus low. The spatial map files are downloaded from this website (https://spatialdata.gov.scot/geonetwork/srv/api/records/f12c3826-4b4b-40e6-bf4f-77b9ed01dc14) and saved into Shape folder. Both open datasets were published in 2019.
