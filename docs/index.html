1 Introduction

Mental health and cardiovascular disease are two of the most pressing public health challenges facing Scotland today, with the recent 2022 census revealing that Mental Health conditions have worsened since 2011. The intricate relationship between these conditions is well-documented, with mental health disorders such as depression and anxiety often acting as both risk factors for and consequences of heart disease. Selective serotonin reuptake inhibitors (SSRIs), widely prescribed for the treatment of mental health disorders, offer valuable insights into trends in mental health management. Meanwhile, heart disease mortality rates provide a way to examine the broader impacts of mental and physical health on public health outcomes.@herman_cell_2014

This assessment explores the intersection of mental health and heart disease mortality in Scotland by visualizing combined trends in SSRI prescriptions for fluoxetine, sertraline, and citalopram. It highlights how mental health and heart disease evolve over time, noting that higher SSRI use in areas of greater socioeconomic deprivation may align with increased heart disease mortality.

Without conducting analysis, this report presents a cohesive and detailed overview of the data, offering foundation for future exploration of these critical public health issues.

# All CSV files listed in the specified directory
ADFiles <- list.files(here("docs","WinterDrugs2015-2023"), pattern = "csv")

# Function defined to read and process each CSV file cohesively
ReadADFiles <- function(file_name) {
  # Extract the year from the file name using lubridate package in tidyverse. This enables plotting with Year as a variable
  year <- str_extract(file_name, "\\d{4}") %>% as.numeric()
  # Read the CSV file, process the data add year column
  data <- read_csv(here("docs","WinterDrugs2015-2023", file_name), show_col_types = FALSE,  col_types = cols(.default = "c")) %>% # Convert all columns to character type to avoid warnings due to inconsistent data types
    mutate(Year = year, PaidQuantity = as.numeric(PaidQuantity)) %>% # Specifically convert paid quantity to numeric format because the data contains numbers, allows for mathematical operations (e.g sum(), mean())
#Filter for specific antidepressants
    filter(str_detect(BNFItemDescription, "MIRTAZAPINE|FLUOXETINE|VENLAFAXINE|AMITRIPTYLINE"),) %>%
    return(data)
}
# This applies the ReadADFiles function to all the CSV files and combines the results, the large amount of data
ADs_Df <- ADFiles %>%
  map_dfr(ReadADFiles)
